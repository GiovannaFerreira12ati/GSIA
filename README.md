## Descrição

O Mission Control AI é um sistema inteligente de monitoramento de missão espacial.

A solução gera dados simulados de temperatura, energia, comunicação e status dos módulos, aplicando regras de alerta e tomada de decisão automática.

Além disso, utiliza o modelo Llama 3.2 1B via Ollama para realizar análises inteligentes da situação operacional da missão.



## Tecnologias Utilizadas

- Python
- Google Colab
- Ollama
- Llama 3.2 1B



## Funcionalidades

- Geração de dados simulados
- Monitoramento de temperatura
- Monitoramento de energia
- Monitoramento de comunicação
- Monitoramento de módulos
- Alertas automáticos
- Tomada de decisão automática
- Análise com Inteligência Artificial
